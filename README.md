# Bech32

[![pub package](https://img.shields.io/pub/v/bech32.svg)](https://pub.dartlang.org/packages/bech32)

An implementation of the [BIP173 spec] for Bech32 address format.

## Examples


## Installing

Add it to your `pubspec.yaml`:

```
dependencies:
  bech32: any
```
